# 下班倒计时动态壁纸生成器

> 输出一个零依赖单文件 HTML，全屏当壁纸用；改顶部 CONFIG 即生成专属版。可作为 Red Skill 发布（粉丝复制装进 AI 生成自己的）。

## 怎么生成（标准流程）
1. **问用户 4 件事**（没说就用默认）：
   - 下班时间（如 18:00）
   - 称呼（打工人/名字，可空）
   - 主题：`neon` 赛博霓虹 / `warm` 暖色治愈 / `mono` 极简黑白
   - 下班后弹幕风格：`hard` 狠话版（骂老板催你走）/ `soft` 温和版（提醒早点回家）
   - （可选）每月发薪日几号、要不要周末倒计时
2. **复制 `assets/wallpaper.html`**，把顶部 `CONFIG` 改成用户的值（offTime/name/theme/barrageStyle/payday/weekend/mochiu）。
3. 输出到用户指定目录（默认项目内或桌面），文件名如 `下班倒计时.html`。
4. **可选预览**：用 Chrome headless 渲染一张 PNG 给用户看（手机 1080×1920 / 桌面 1920×1080）：
   ```bash
   "/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless --disable-gpu \
     --window-size=1080,1920 --screenshot=preview.png "file://$(pwd)/下班倒计时.html"
   ```

## CONFIG 字段
- `offTime` 下班时间 24h ｜ `name` 称呼 ｜ `theme` neon/warm/mono
- `barrageStyle` hard/soft ｜ `payday` 发薪日(0关) ｜ `weekend` 周末倒计时
- `workHours` 工作时长(进度条用) ｜ `mochiu[]` 摸鱼文案

## 怎么当壁纸（HTML 不能直接设壁纸，要讲清楚）
- **Mac 桌面**：装 Plash，把 HTML 网址/路径粘进去
- **Win 桌面**：装 Lively Wallpaper，添加网页
- **手机锁屏/动态壁纸**：浏览器全屏打开 → 录屏 10–15 秒 → 存视频 → 设为动态壁纸/锁屏
- HTML 内已内置「?」按钮弹出这套说明

## 内置功能（基于真实成品）
实时倒计时 / 粒子连线 / 今日工作进度条 / 下班后红色告警+抖动横幅+弹幕 / 周末·发薪 chips / 摸鱼文案轮播 / F11·双击全屏。

## 做成小红书内容时（成果型·涨粉）
- 选题："不会编程，我用 AI 做了个下班倒计时壁纸"——展示成果+过程，比讲技巧涨粉。
- Red Skill 植入站内话术："放笔记里了，复制一下就能装进你的 AI，改成你自己的下班时间。"
- 合规：小红书笔记里**别提具体外部域名**（站外导流红线）；弹幕"狠话版"自用可以，公开内容可切"温和版"更稳；发布勾「AI 辅助创作」。
