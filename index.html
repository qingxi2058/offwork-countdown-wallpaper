🖥️ 怎么把它变成壁纸

它是个全屏网页，三种用法：

🍎 Mac 桌面
  1. App Store 装免费的 Plash
  2. Plash → Preferences → 把本页网址粘进 URL
🪟 Windows 桌面
  1. Microsoft Store 装免费的 Lively Wallpaper
  2. + 添加 → 网页 → 粘网址 → 设为壁纸
📱 手机锁屏 / 动态壁纸
  1. 手机/电脑浏览器全屏打开本页
  2. 录屏 10–15 秒 → 存成视频
  3. 设为手机动态壁纸 / 锁屏
